# Week_2-javascript

# 📦 Gestión de Datos con Objetos, Sets y Maps

Proyecto desarrollado en JavaScript para practicar estructuras de datos y manejo de información.

---

# 🧠 Temas trabajados

* Objetos en JavaScript
* Sets
* Maps
* Validaciones
* Métodos de objetos
* Recorridos con bucles

---

# 📁 Estructura del proyecto

```bash
gestion_datos.js
```

---

# ✅ Características del proyecto

## 🧩 Manejo de Objetos

Cada producto contiene:

* ID
* Nombre
* Precio

Ejemplo:

```js
{
   id: 1,
   nombre: "Laptop",
   precio: 2500
}
```

---

## 🔁 Uso de Set

El proyecto utiliza `Set` para:

* eliminar datos repetidos
* agregar nuevos valores
* verificar existencia de datos
* eliminar elementos

---

## 🗂️ Uso de Map

El proyecto usa `Map` para relacionar:

```txt
Categoría → Producto
```

Ejemplo:

```txt
Tecnología → Laptop
```

---

# 🔄 Métodos utilizados

En el proyecto se implementaron:

* `for...in`
* `for...of`
* `forEach()`
* `Object.keys()`
* `Object.values()`
* `Object.entries()`

---

# ✅ Validaciones implementadas

Se verifican:

* productos sin ID
* productos sin nombre
* precios inválidos

---

# 🎯 Objetivo del proyecto

Comprender cómo organizar y manipular datos utilizando estructuras avanzadas de JavaScript de manera sencilla y ordenada.

---

# 💻 Resultado esperado

El programa muestra en consola:

* lista de productos
* números únicos usando Set
* categorías y productos usando Map
* validaciones de datos
